引言

优化模型性能的一些途径(并非全部有效，需要根据项目特点进行选择):

  • 获取更多的训练数据 (有时候无效)
  • 调整特征数量 (增大/减少 特征数量)
  • 增加多项式特征 (例如: x12,x22,x1x2,x_1^2 , x_2^2 , x_1x_2 ,x12​,x22​,x1​x2​, etc.)
  • 调整正则化参数 λ\lambdaλ

机器学习诊断 工作比较重要，可以协助找到模型 优化的方向

评估假设

生成训练集与测试集

将一组数据集 data set 按照 7:3 的比例分成 训练集 training set 与 测试集 test set.

数据集名称 英文名称 占比
训练集 training set 70%
测试集 test set 30%

数据集排序 最好是 无序 、 随机 的

模型选择 与 数据集划分

  1. 训练集误差不能当做模型的性能评价指标(不能对实际泛化能力做出很好的判断)
  2. 如果使用 测试集 作为假设函数的验证数据的话，也存在 实际泛化能力 判断的缺陷，不能很好的反应实施情况。

解决方案：增加 交叉验证集 cross validation set 简称 cv

数据集名称 英文名称 占比
训练集 training set 60%
交叉验证集 cross validation set 20%
测试集 test set 20%

可得到各个集的误差:

  • 训练集误差:
    Jtrain(θ)=12mtrain ∑i=1mtrain(hθ(xtrain(i))−ytrain(i))2J_{train}(\theta) = \dfrac{1}{2m_{train}} \ \sum^{m_{train}}_{i=1}\Bigl(h_{\theta}(x^{(i)}_{train}) - y^{(i)}_{train} \Bigr)^2Jtrain​(θ)=2mtrain​1​ ∑i=1mtrain​​(hθ​(xtrain(i)​)−ytrain(i)​)2

  • 交叉验证集误差:
    Jcv(θ)=12mcv ∑i=1mcv(hθ(xcv(i))−ycv(i))2J_{cv}(\theta) = \dfrac{1}{2m_{cv}} \ \sum^{m_{cv}}_{i=1}\Bigl(h_{\theta}(x^{(i)}_{cv}) - y^{(i)}_{cv} \Bigr)^2Jcv​(θ)=2mcv​1​ ∑i=1mcv​​(hθ​(xcv(i)​)−ycv(i)​)2

  • 测试集误差:
    Jtest(θ)=12mtest ∑i=1mtest(hθ(xtest(i))−ytest(i))2J_{test}(\theta) = \dfrac{1}{2m_{test}} \ \sum^{m_{test}}_{i=1}\Bigl(h_{\theta}(x^{(i)}_{test}) - y^{(i)}_{test} \Bigr)^2Jtest​(θ)=2mtest​1​ ∑i=1mtest​​(hθ​(xtest(i)​)−ytest(i)​)2

利用交叉验证集选择模型假设，可以节省测试集，这样就可以利用测试集来评测模型的泛化能力了

判断方法：

  • 左侧是 高偏差 欠拟合状态

    Jtrain(θ)J_{train}(\theta)Jtrain​(θ) 较大，同时 Jcv(θ)≈Jtrain(θ)J_{cv}(\theta) \approx J_{train}(\theta)Jcv​(θ)≈Jtrain​(θ)

  • 右侧是 高方差 过拟合状态

    Jtrain(θ)J_{train}(\theta)Jtrain​(θ) 较小，同时 Jcv(θ)>>Jtrain(θ)J_{cv}(\theta) >> J_{train}(\theta)Jcv​(θ)>>Jtrain​(θ)

正则化 偏差 与 方差

正则化参数 λ\lambdaλ 对训练的影响

描述 英文名称 状态 正则化参数
高偏差 Hight bias 欠拟合 λ\lambdaλ 非常大 (λ=10000\lambda = 10000λ=10000)，θ1≈0,θ2≈0,...,hθ(x)≈θ0\\ \theta_1 \approx 0,\theta_2 \approx 0,...,h_\theta(x) \approx \theta_0θ1​≈0,θ2​≈0,...,hθ​(x)≈θ0​

选择合适的正则化参数 λ\lambdaλ

正则化只用于训练时的误差函数，评测使用的误差函数不加正则化

  • 训练时使用的带有正则化的误差函数:
    J(θ)=12m ∑i=1m(hθ(x(i))−y(i))2+λ2m ∑j=1mθj2J(\theta) = \dfrac{1}{2m} \ \sum^{m}_{i=1}\Bigl(h_{\theta}(x^{(i)}) - y^{(i)} \Bigr)^2 + \dfrac{\lambda}{2m} \ \sum^m_{j=1} \theta^2_jJ(θ)=2m1​ ∑i=1m​(hθ​(x(i))−y(i))2+2mλ​ ∑j=1m​θj2​

  • 训练集误差:
    Jtrain(θ)=12mtrain ∑i=1mtrain(hθ(xtrain(i))−ytrain(i))2J_{train}(\theta) = \dfrac{1}{2m_{train}} \ \sum^{m_{train}}_{i=1}\Bigl(h_{\theta}(x^{(i)}_{train}) - y^{(i)}_{train} \Bigr)^2Jtrain​(θ)=2mtrain​1​ ∑i=1mtrain​​(hθ​(xtrain(i)​)−ytrain(i)​)2

  • 交叉验证集误差:
    Jcv(θ)=12mcv ∑i=1mcv(hθ(xcv(i))−ycv(i))2J_{cv}(\theta) = \dfrac{1}{2m_{cv}} \ \sum^{m_{cv}}_{i=1}\Bigl(h_{\theta}(x^{(i)}_{cv}) - y^{(i)}_{cv} \Bigr)^2Jcv​(θ)=2mcv​1​ ∑i=1mcv​​(hθ​(xcv(i)​)−ycv(i)​)2

  • 测试集误差:
    Jtest(θ)=12mtest ∑i=1mtest(hθ(xtest(i))−ytest(i))2J_{test}(\theta) = \dfrac{1}{2m_{test}} \ \sum^{m_{test}}_{i=1}\Bigl(h_{\theta}(x^{(i)}_{test}) - y^{(i)}_{test} \Bigr)^2Jtest​(θ)=2mtest​1​ ∑i=1mtest​​(hθ​(xtest(i)​)−ytest(i)​)2

判断方法:

  • λ\lambdaλ 越小: 高方差 模型处于 过拟合 状态
  • λ\lambdaλ 越大: 高偏差 模型处于 欠拟合 状态

学习曲线 Learning curves

  • 训练集越小，训练集误差越小；训练集越大，训练集误差越大。
  • 训练集越小，交叉训练集误差越大；训练集越大，交叉训练集误差越小。

高偏差情况 High bias

交叉训练集与训练集会随着训练数据的增加变得越来越接近

特点是 数据量 mmm 增大到一定程度后，Jcv(θ)J_{cv}(\theta)Jcv​(θ) 与 Jtrain(θ)J_{train}(\theta)Jtrain​(θ) 相对较大 且 相对平直不再变化
这样的情况下，增加数据集没有什么作用

高方差情况 High vairance

交叉训练集与训练集误差之间存在一个比较大的距离 (gap)

特点是 数据量 mmm 增大到一定程度后，Jcv(θ)J_{cv}(\theta)Jcv​(θ) 与 Jtrain(θ)J_{train}(\theta)Jtrain​(θ) 之间还是相差比较远，Jcv(θ)J_{cv}(\theta)Jcv​(θ) 误差较大，Jtrain(θ)J_{train}(\theta)Jtrain​(θ) 误差较小
这样的情况下，增加数据集会有一定的帮助

总结

方法 解决问题
获取更多的训练数据 (有时候无效) 解决高方差
减少特征数量 解决高方差
增大特征数量 解决高偏差
增加多项式特征 (例如: x12,x22,x1x2,x_1^2 , x_2^2 , x_1x_2 ,x12​,x22​,x1​x2​, etc.) 解决高偏差
增大正则化参数 λ\lambdaλ 解决高偏差
减小正则化参数 λ\lambdaλ 解决高方差

神经网络

  • 小型的神经网络 容易 欠拟合
  • 大型的神经网络 容易 过拟合 ( 巧用正则化 λ\lambdaλ )
