误差分析

推荐的方法：

  1. 使用一个相对简单粗暴的算法快速启动项目，然后使用交叉验证集测试；
  2. 绘制学习曲线(learning curves)，然后根据曲线决定优化方案(更多数据/更多特征);
  3. 误差分析: 观察错误的结果，总结出特征，然后优化系统；
    • 注意频率多的错误，并找出对应方案
  4. 数值估计
    • 使用一些方法，完善算法 ( 词干统计 etc. )
    • 利用 交叉验证机 低成本试错 (观察 误差变化)

不对称性分类的误差分析

准确率 Precision / 召回率 Recall

针对一个类别 ( 例如 y=1y=1y=1 ) 进行预测

预测 \ 实际 实际 1 实际 0
预测 1 真正类 TP 假正类 FP
预测 0 假负类 FN 真负类 TN

准确率 =TPTP+FP=\dfrac{TP}{TP+FP}=TP+FPTP​ ( 又名 查准率 )

Precision 准确率 = 真实的正类 / 所有预测的正类

召回率 =TPTP+FN=\dfrac{TP}{TP+FN}=TP+FNTP​ ( 又名 查全率 )

Recall 召回率 = 真实的正类 / 所有正类

准确率与召回率的权衡

调节阈值 thresholdthresholdthreshold 可以改变 hθ(x)⩾ thresholdh_\theta(x) \geqslant \ thresholdhθ​(x)⩾ threshold,然后调节 准确率 与 召回率

  • 阈值低 - 高召回率，低准确率
  • 阈值高 - 高准确率，低召回率

利用 召回率-准确率曲线 找到平衡点
