过拟合问题

拟合中出现的问题

举例：房屋价格 线性回归

  • 橙色 欠拟合 : θ0+θ1x\theta_0 + \theta_1xθ0​+θ1​x ( 高偏差 high bias )
  • 蓝色 正合适 : θ0+θ1x+θ2x2\theta_0 + \theta_1x+\theta_2x^2θ0​+θ1​x+θ2​x2
  • 红色 过拟合 : θ0+θ1x+θ2x2+θ3x3++θ4x4\theta_0 + \theta_1x+\theta_2x^2+\theta_3x^3++\theta_4x^4θ0​+θ1​x+θ2​x2+θ3​x3++θ4​x4 ( 高方差 high variance )

随着 特征 的增加，越容易出现 过拟合

解决过拟合

例如目前有很多特征，目前解决的方法:

  1. Reduce number of features. 减少特征数量:
    • 人工选取重要的特征；
    • 模型选择算法；
  2. Regularization. 正则化:
    • 保留所有特征，但是为每个参数 θj\theta_jθj​ 增加权重；
    • 在存在大量特征的任务中表现不错，每一个特征都可以对预测 yyy 有所帮助；

代价函数

观察上一章房屋价格预测曲线图：

  • 蓝色 正合适 : θ0+θ1x+θ2x2\theta_0 + \theta_1x+\theta_2x^2θ0​+θ1​x+θ2​x2
  • 红色 过拟合 : θ0+θ1x+θ2x2+θ3x3+θ4x4\theta_0 + \theta_1x+\theta_2x^2+\theta_3x^3+\theta_4x^4θ0​+θ1​x+θ2​x2+θ3​x3+θ4​x4

如果想解决过拟合(过多的抖动)，需要 惩罚 参数 θ3,θ4\theta_3,\theta_4θ3​,θ4​ (控制高阶的系数), 让他们足够 小。

那么代价函数将改写为: minθ  12m∑i=1m(hθ(x(i))−y(i))2+1000×θ32+1000×θ42\underset{\theta}{min} \ \ \dfrac{1}{2m}\displaystyle\sum^m_{i=1}(h_\theta(x^{(i)})-y^{(i)})^2 \color{blue}{+1000\times \theta_3^2+1000\times \theta_4^2}θmin​  2m1​i=1∑m​(hθ​(x(i))−y(i))2+1000×θ32​+1000×θ42​

由于 θ3,θ4\theta_3,\theta_4θ3​,θ4​ 前面加了很大的系数，所以为了最小化代价函数，θ3,θ4\theta_3,\theta_4θ3​,θ4​ 只能接近于 000
多项式 θ0+θ1x+θ2x2+θ3x3+θ4x4\theta_0 + \theta_1x+\theta_2x^2+\theta_3x^3+\theta_4x^4θ0​+θ1​x+θ2​x2+θ3​x3+θ4​x4 将会接近 θ0+θ1x+θ2x2\theta_0 + \theta_1x+\theta_2x^2θ0​+θ1​x+θ2​x2
这样会得到一个更好的假设，从而可以更好的拟合

正则化

将参数 θ0,θ1,...,θn\theta_0,\theta_1,...,\theta_nθ0​,θ1​,...,θn​ 尽可能的缩小

  • 可以得到更为简化的假设
  • 减少过拟合

改写代价函数的方程: minθ  J(θ)=12m[∑i=1m(hθ(x(i))−y(i))2+λ∑j=1nθj2]\underset{\theta}{min} \ \ J(\theta) = \dfrac{1}{2m}\Bigl[\displaystyle\sum^m_{i=1}(h_\theta(x^{(i)})-y^{(i)})^2 + \lambda\displaystyle\sum^n_{j=1}\theta^2_j \Bigr]θmin​  J(θ)=2m1​[i=1∑m​(hθ​(x(i))−y(i))2+λj=1∑n​θj2​]

  • 第一项 ∑i=1m(hθ(x(i))−y(i))2\sum^m_{i=1}(h_\theta(x^{(i)})-y^{(i)})^2∑i=1m​(hθ​(x(i))−y(i))2 是为了让假设方程更加 拟合数据
  • 第二项 ∑j=1nθj2\sum^n_{j=1}\theta^2_j∑j=1n​θj2​ 是为了让参数尽可能的 小
  • λ\lambdaλ 是调节两个部分的平衡参数 (regularization parameter 正则化参数)

如果 λ\lambdaλ 设置过大，则将会忽略所有参数 θj,j=1,2,...,n\theta_j,j=1,2,...,nθj​,j=1,2,...,n , 假设将变为 欠拟合 状态
